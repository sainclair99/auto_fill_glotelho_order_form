Comment ça marche ?

2 - Ensuite cliquez sur le bouton valider la commande

3 - Un message de confirmation sera envoyé a ce numéro pour le paiement. Continuez en entrant votre code secret.

Si vous n'avez pas reçu de message, veuillez svp composer le #150# pour Orange ou *126# pour MTN afin de confirmer la transaction en cours. Pour des raisons de sécurité, rassurez-vous bien que le nom de l'initiateur de cette transaction soit bel et bien Mesomb.
